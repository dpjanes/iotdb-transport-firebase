# Installation and Configuration

## Install

    $ npm install -g homestar ## may require sudi
    $ npm install iotdb-transport-firebase

## Configure

    $ homestar set /transports/iotdb-transport-firebase/initd/host 'https://yourname.firebaseIO.com/'
